# Politica de privacidade

`Leitor de Artigos` nao coleta, nao armazena e nao transmite dados pessoais para servidores externos.

URL publica recomendada para uso nas lojas:

- `https://github.com/jairslima/leitor-de-artigos/blob/master/PRIVACY.md`

## O que a extensao faz

A extensao atua localmente no navegador para:

- analisar a estrutura HTML da pagina aberta
- identificar o bloco principal de leitura
- remover ruido visual comum, como barras laterais e banners
- exibir um modo de leitura limpo na propria aba

## Dados

A extensao:

- nao cria conta de usuario
- nao coleta nome, email, telefone ou documento
- nao envia historico de navegacao para servidor
- nao usa analytics externos
- nao vende ou compartilha dados

## Permissoes

A permissao de acesso amplo a paginas existe para que a extensao possa funcionar em diferentes sites de artigo. O processamento acontece localmente no navegador do usuario.

## Contato

Se esta politica precisar ser adaptada antes da publicacao, atualizar este arquivo e a URL publica correspondente usada nas lojas.
